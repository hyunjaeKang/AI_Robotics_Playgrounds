# LeRobot

### Setup a conda environment

 ```
 conda create -y -n lerobot python=3.10
 conda activate lerobot
 conda install ffmpeg -c conda-forge

 pip install 'lerobot[all]'
 pip install ipykernel
 pip install ipywidgets
 ```
